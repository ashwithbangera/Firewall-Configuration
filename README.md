# Firewall Configuration & Testing

## Objective
To configure and test firewall rules using UFW on Kali Linux and understand firewall management.

## Tools Used
- Kali Linux
- UFW (Uncomplicated Firewall)

## Task Summary
- Enabled UFW firewall
- Allowed essential ports (22, 80, 443)
- Blocked insecure port (23)
- Blocked malicious IP address
- Enabled and monitored firewall logs
- Tested connectivity for allowed and blocked rules

## Screenshots
All configuration and testing screenshots are available in the screenshots folder.

## Outcome
This task helped in understanding firewall rules, traffic filtering, logging, and real-world firewall management.
